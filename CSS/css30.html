WEB DEVELOPMENT

HTML

HTML is web development language used to create the web pages, It is also callled as formatting language. without HTML we can't create the web pages.

CSS

CSS stands for Cascading Style Sheet bascially it is used to style the web pages created by HTML. It is used to design the web pages and give them attractive look.

JS

JS stands for JavaScript where it is used to give the dynamic behaviour to our web pages created by using HTML and CSS . Using JS we can create event handler as well. There are two types of JS i.e. Client Side JS and Server Side JS.
